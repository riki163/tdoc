# Класс AdvancedCalculator ## Описание Класс `AdvancedCalculator` представляет собой расширенный калькулятор с поддержкой различных математических операций, включая сложение, вычитание, умножение, деление, возведение в степень, извлечение квадратного корня, синус, косинус, тангенс и логарифм. ## Методы ### \_\_init\_\_ (self) - **Описание:** Конструктор класса. - **Параметры:** Нет параметров. - **Возвращаемое значение:** Создает экземпляр класса `AdvancedCalculator` с инициализированными переменными `result` (результат вычислений) и `history` (история операций). ### add (self, x, y) - **Описание:** Выполняет сложение двух чисел и сохраняет результат в переменной `result`. - **Параметры:** - `x` (float): Первое число. - `y` (float): Второе число. - **Возвращаемое значение:** Возвращает результат сложения. ### subtract (self, x, y) - **Описание:** Выполняет вычитание одного числа из другого и сохраняет результат в переменной `result`. - **Параметры:** - `x` (float): Уменьшаемое число. - `y` (float): Вычитаемое число. - **Возвращаемое значение:** Возвращает результат вычитания. ### multiply (self, x, y) - **Описание:** Выполняет умножение двух чисел и сохраняет результат в переменной `result`. - **Параметры:** - `x` (float): Первое число. - `y` (float): Второе число. - **Возвращаемое значение:** Возвращает результат умножения. ### divide (self, x, y) - **Описание:** Выполняет деление одного числа на другое и сохраняет результат в переменной `result`. Проверяет наличие деления на ноль. - **Параметры:** - `x` (float): Делимое число. - `y` (float): Делитель. - **Возвращаемое значение:** Возвращает результат деления или сообщение о невозможности деления на ноль. ### power (self, x, y) - **Описание:** Возводит число в указанную степень и сохраняет результат в переменной `result`. - **Параметры:** - `x` (float): Основание степени. - `y` (float): Показатель степени. - **Возвращаемое значение:** Возвращает результат возведения в степень. ### square_root (self, x) - **Описание:** Извлекает квадратный корень из числа и сохраняет результат в переменной `result`. Проверяет, что число не отрицательное. - **Параметры:** - `x` (float): Число, из которого извлекается квадратный корень. - **Возвращаемое значение:** Возвращает результат извлечения квадратного корня или сообщение о невозможности извлечения из отрицательного числа. ### sin (self, x) - **Описание:** Вычисляет синус угла в градусах и сохраняет результат в переменной `result`. - **Параметры:** - `x` (float): Угол в градусах. - **Возвращаемое значение:** Возвращает результат вычисления синуса. ### cos (self, x) - **Описание:** Вычисляет косинус угла в градусах и сохраняет результат в переменной `result`. - **Параметры:** - `x` (float): Угол в градусах. - **Возвращаемое значение:** Возвращает результат вычисления косинуса. ### tan (self, x) - **Описание:** Вычисляет тангенс угла в градусах и сохраняет результат в переменной `result`. - **Параметры:** - `x` (float): Угол в градусах. - **Возвращаемое значение:** Возвращает результат вычисления тангенса. ### log (self, x, base=10) - **Описание:** Вычисляет логарифм числа по указанному основанию и сохраняет результат в переменной `result`. - **Параметры:** - `x` (float): Число, логарифм которого вычисляется. - `base` (float, по умолчанию 10): Основание логарифма. - **Возвращаемое значение:** Возвращает результат вычисления логарифма. ### get_history (self) - **Описание:** Возвращает историю операций, сохраненную в переменной `history`. - **Параметры:** Нет параметров. - **Возвращаемое значение:** Список строк, представляющих операции из истории. ## Функция run_calculator() ### Описание Функция `run_calculator` представляет собой текстовый интерфейс для взаимодействия с калькулятором. ### Запуск Для запуска калькулятора необходимо вызвать функцию `run_calculator()`. ### Интерфейс - Выводит меню с доступными операциями. - Позволяет выбирать операции с вводом соответствующего номера. - В зависимости от выбранной операции просит ввести соответствующие числа или угол. - Выводит результат операции или сообщение об ошибке. - Позволяет просматривать историю операций.
